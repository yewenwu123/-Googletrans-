# -Googletrans-
如果是在中国大陆的话,使用下面的代码必须使用翻墙，而且还得是全局。翻墙一般用clash的tun模式比较适合。
这个脚本使用了Googletrans库，它可以读取一个文本文件中的内容，将其翻译为指定的目标语言
